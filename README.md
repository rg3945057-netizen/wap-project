# 🎌 Anime Facts Explorer

## 📌 Project Overview

Anime Facts Explorer is a simple and interactive web application that allows users to explore interesting facts about various anime using a public API. The project is built using HTML, CSS, and JavaScript, focusing on API integration and responsive UI design.

---

## 🎯 Purpose

The purpose of this project is to:

* Practice working with REST APIs
* Improve frontend development skills
* Build an engaging and user-friendly interface
* Learn asynchronous JavaScript (fetch, promises)

---

## 🔗 API Used

This project uses the **Anime Facts REST API**:

👉 https://chandan-02.github.io/anime-facts-rest-api/

The API provides:

* List of anime
* Facts related to each anime
* Random anime facts

---

## ✨ Features

* 🔍 Search anime by name
* 📜 Display anime facts dynamically
* 🎲 Get random anime facts
* ❤️ Save favorite facts (local storage)
* 🌙 Dark mode toggle
* 📱 Fully responsive design
* ⚡ Smooth loading animations
* 🚫 Error handling for invalid inputs

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* LocalStorage

## ⚙️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/anime-facts-explorer.git
   ```

2. Navigate to the project folder:

   ```bash
   cd anime-facts-explorer
   ```

3. Open `index.html` in your browser
